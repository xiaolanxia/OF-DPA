OF-DPA
======

OpenFlow - Data Plane Abstraction

Overview
========

Broadcom's OpenFlow Data Plane Abstraction (OF-DPA) is an application software
component that implements an adaptation layer between OpenFlow and the
Broadcom Silicon SDK. OF-DPA enables scalable implementation of OpenFlow 1.3.4
on Broadcom switch devices.

Documentation
=============

Detailed documentation can be found at http://Broadcom-Switch.github.io/of-dpa/doc/html/index.html.
